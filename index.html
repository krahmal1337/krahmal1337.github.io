FAQ

Common questions

What cheats does this work with?

Neverlose (удобный/HvH), Skeet (топ-1 HvH), Onetap (старая залупа). Windows 10/11, .NET 8.

Is this really free?

Yes. No subscriptions, no paywalls, no "premium" tiers. The project is MIT-licensed and always will be.

How does auto-update work?

On launch, nadeloader checks for new releases on GitHub and only downloads the files that changed.

Can I contribute?

Absolutely! Pull requests, issues, and feature suggestions are welcome. Check the GitHub repo to get started.

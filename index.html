FAQ

Common questions

What systems does this run on?

Windows 10 and 11, x64 and ARM64. Requires .NET 8 — the loader will prompt you if it's missing.

Do I need to install anything?

No. Unzip the archive, run the executable — profiles and settings live right next to it.

How does auto-update work?

On launch, nadeloader checks the server version and only downloads the files that changed.

Can I cancel a Pro subscription?

Yes, anytime from your account — access stays active until the end of the paid period.
